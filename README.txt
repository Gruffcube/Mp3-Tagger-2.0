Introduction and use case ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This MP3 tagger is best used for adding album covers, ordering, and 
other metadata to MP3 files that have been ripped from youtube and 
coverted (this software does not convert or download anything - ideally 
use software such as jdownloader 2 and VLC media player for this). The 
best use case for this software, is adding metadata to albums 
specifically. Give the data as a folder containing all of the mp3 
files, as well as the album cover given as a JPEG. This software does 
not find files in subdirectories.
